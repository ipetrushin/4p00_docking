# Комплект программ для докинга лигандов к рецептору 4P00 https://www.rcsb.org/structure/4P00

# Ход работы:
- установить MGLTools версии 1.5.7 (в скриптах прописаны пути к файлам по умолчанию, если у вас они отличаются, измените)
- найти 5-10 веществ из класса флавоноидов/алкалоидов в базе ZINC15, скачать их структуры в формате SDF
- ссылки на примеры веществ: https://zinc15.docking.org/substances/ZINC000230101161/ https://zinc15.docking.org/substances/ZINC000100829264 https://zinc15.docking.org/substances/ZINC000100890138
- с помощью онлайн-инструментов конвертировать формат файла из SDF в PDB http://www.cheminfo.org/Chemistry/Cheminformatics/FormatConverter/index.html
- полученные PDB файлы со структурой молекул лиганда скопировать в основную папку (там же, где и файл vina.exe)
- запустить командный интерпретатор cmd в этой папке
- для каждого файла со структурой молекулы лиганда запустить скрипт prepare_ligand.cmd указав в качестве параметра имя файла без расширения (например, test). При этом в папке должен появиться аналогичный файл с расширением pdbqt, переместите его в папку in
- затем запустить скрипт dock_ligand.cmd указав в качестве параметра имя файла без расширения (например, test)
- запустится программа моделирования, после завершения вы получите файл журнала в папке log и файл с моделями в папке out, это и есть результат работы
